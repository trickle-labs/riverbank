# Tutorials

Tutorials are end-to-end walkthroughs that produce a working result. Each follows a realistic scenario, shows every command, and explains what happened.

| Tutorial | What you'll build |
|----------|------------------|
| [Compile a policy corpus](compile-a-policy-corpus.md) | A knowledge graph from documentation using competency questions |
| [Compile a runbook corpus](compile-a-runbook-corpus.md) | Procedural knowledge with step sequences and error-handling paths |
| [Multi-tenant setup](multi-tenant-setup.md) | Isolated tenants with RLS and per-tenant named graphs |
| [Render a MkDocs site](render-a-mkdocs-site.md) | Auto-generated knowledge pages published via GitHub Actions |

!!! tip
    Tutorials are not reference material — they show one path through the system. For exhaustive flag documentation, see the [reference](../reference/index.md).
